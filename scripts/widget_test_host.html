Concierge widget — host page simulator

This page emulates a customer's website. The chat bubble should appear in the lower-right corner.

1. How a website owner installs the widget

Paste the snippet from the Admin panel's Embed Snippet page:

<script src="http://localhost:8000/widget.js" data-widget-id="YOUR_PUBLIC_WIDGET_ID" async></script>

2. Test instructions

  1. Open the Admin panel → Embed Snippet, copy the snippet.
  2. Replace the script tag below with the copied snippet.
  3. Serve this file from an origin in the widget's allowed_origins list (e.g. cd Concierge/scripts && python3 -m http.server 5500 and then add http://localhost:5500 to allowed_origins via the Admin panel).
  4. Click the bubble, send a message, and watch the assistant reply.
